------------------------------------------------------
<a href="https://v-project.my.id"> Demo Website </a>
------------------------------------------------------
Quick Use (Laravel 10)(PHP > 8.2)(LIVEWIRE 3)
------------------------------------------------------
- clone this repository
- composer update
- cp .env.example .env
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- npm install
- npm run dev / npm run build && php artisan serve
